# 🧳✈️ Projeto Réfugio 💰🪪

Serviço de turismo para nomades digitais. [APP]

▶️ ## Execução

Para o projeto funcionar será necessário que você tenha instalado em sua máquina o Nodejs em sua versão mais estável: <br>
Para adquirir o Nodejs acesse: https://nodejs.org/en/download/ <br> <br>

Será necessário o instalador de pacotes npm ou yarn e também um dispostivo mobile android ou ios. <br>
Obs: se você instalou o Nodejs através do link acima, não se preocupe, por padrão já vem disponível o npm) <br>

Para vizualizar o APP em modo de debug será necessário que em seu dispositivo possua o aplicativo da Expo para realizar o download acesse: 

No Android: https://play.google.com/store/apps/details?id=host.exp.exponent <br>
No IOS: https://apps.apple.com/br/app/expo-go/id982107779

## Passo 01
 Copie o projeto aqui do git para a sua máquina e acesse a pasta do projeto:

Acessando a pasta do projeto pela linha de comando:
```bash
  cd refugio
```
    
 ## Passo 02
 
  Execute o projeto com o seguinte comandoa através do terminal:
 ```bash
  npm run start
```

 ## Passo 03
 Será exibido em sua tela um QRCode, faça a leitura do mesmo com o seu dispositivo e acesse o link, automaticamente será aberto o aplicativo do Expo executando este projeto.
